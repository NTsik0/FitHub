# ⚡ FITNESS HUB

**Your journey starts here.**

A modern, responsive fitness platform UI built with pure HTML & CSS.  
Designed to connect users with gyms, trainers, and personalized fitness plans across Georgia.

---

## 🏋️ Overview

**FITNESS HUB** is a front-end project that showcases a complete fitness ecosystem:

- 🏢 Gym discovery  
- 👨‍🏫 Trainer profiles  
- 📅 Booking & scheduling system  
- 🗺️ Interactive map integration  
- 📝 Registration & contact form  

Built with a strong focus on **clean design**, **user experience**, and **responsive layout**.

---

## ✨ Features

- ⚡ Split-screen hero section (Gyms vs Trainers)  
- 📍 Featured fitness centers with ratings  
- 👥 Instructor showcase  
- 📅 Weekly training schedule  
- 🗺️ Embedded Google Maps view  
- 📝 Booking & registration form  
- 📱 Mobile-friendly navigation (hamburger menu)  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (custom styling, no frameworks)  
- Google Fonts (Bebas Neue, Barlow)  

---

## 🚀 Getting Started

1. Clone the repository:
   git clone https://github.com/NTsik0/FitHub.git

2. Open `index.html` in your browser.

No build tools. No dependencies. Just open and go.
