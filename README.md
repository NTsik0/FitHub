# ⚡ FitHub | Your Peak Potential


[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com)

**FitHub** is a premium, split-interface web platform designed to bridge the gap between Georgian fitness seekers, high-end gym facilities, and expert personal trainers. 

Developed as a modern solution for the Georgian fitness market, FitHub offers a streamlined "dual-track" experience: helping users find the perfect place to sweat or the perfect professional to guide them.

---

## 🚀 Vision
The goal of FitHub is to centralize the fitness ecosystem in Georgia. Whether you are looking for a high-intensity CrossFit box in Tbilisi or a yoga instructor in Batumi, FitHub provides the digital infrastructure to discover, book, and track your progress.

## ✨ Features

### 🏢 For Users & Gym-Goers
*   **Split-Hero Navigation:** A high-impact landing page that directs users immediately to either "Places" (Gyms) or "People" (Instructors).
*   **Gym Finder:** Advanced filtering for locations like *Rule Fitness* or *Olympia*, including ratings, amenities, and 24/7 access status.
*   **Trainer Marketplace:** Profiles for certified experts specializing in Fat Loss, Strength, and Endurance.
*   **Glassmorphism Booking:** A modern, blur-effect consultation form for seamless session scheduling.

### 🛠 Technical Stack
*   **Frontend:** HTML5, CSS3 (Custom Flexbox/Grid layouts), and React integration.
*   **Styling:** Modern typography via Google Fonts (Bebas Neue & Barlow) and Font Awesome 6 icons.
*   **Responsiveness:** Fully fluid design optimized for mobile, tablet, and desktop performance.
*   **UI/UX:** High-contrast "Dark Mode" aesthetic with red accent highlights for an energetic brand feel.

## 📸 Preview
> *The landing page features a unique "Curtain Transition" hero section that allows users to interact with two different paths of the fitness journey simultaneously.*

## 🛠 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/NTsik0/FitHub.git](https://github.com/NTsik0/FitHhub.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd FitHhub
    ```
3.  **Open the project:**
    Simply open `index.html` in your preferred browser or use Live Server in VS Code.
